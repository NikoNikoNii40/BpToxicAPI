Privacy Policy — Toxic Text Detector

Last updated: 2026-05-09

Overview

Toxic Text Detector is a Chrome extension that helps users detect potentially toxic text on web pages. The extension works in a local-by-default mode and may optionally use a remote API if the user enables it in settings.

Data We Process

  • Website content (page text): The extension reads visible text from the current page in order to analyze it for toxicity. This text is processed to produce toxicity scores and UI feedback (e.g., highlighting/blur).
  • Extension settings: We store user configuration (e.g., strictness, enabled labels, selected mode, UI preferences) using Chrome storage.
  • Optional API token: If the user enables remote mode and provides a token, the token is stored in Chrome storage and sent only to the configured API endpoint for authentication.

What We Do NOT Collect

  • No names, email addresses, phone numbers, or other direct identifiers
  • No payment or financial information
  • No passwords or authentication data (other than the optional API token you provide)
  • No location data
  • No web browsing history tracking
  • No sale of user data

Local Mode (Default)

In local mode, text analysis happens on your device in the browser. Page text is not sent to any server.

Remote Mode (Optional)

If you enable remote mode, the extension sends the text you choose to analyze to the remote API endpoint you configured (for example, a Cloud Run service). The API returns toxicity scores.

The remote API is designed to process text in memory for inference and return results. It is not intended to store the submitted text. Basic operational logs may exist (e.g., request timing/status), but the service is not designed to persist page content as a dataset.

How We Use the Data

  • To provide toxicity detection and display results in the extension UI
  • To save your preferences so the extension works consistently across sessions
  • To authenticate requests to the remote API if you enabled remote mode and configured a token

Data Sharing

We do not sell user data. We do not share user data with third parties for advertising or profiling. If you enable remote mode, text is sent only to the API endpoint you configured for inference.

Data Retention

  • Extension settings and optional token are stored until you change them or uninstall the extension.
  • Page text is processed for analysis and not intended to be stored permanently.

Security

Remote requests use HTTPS when configured with an HTTPS endpoint. You can keep remote mode disabled to ensure processing stays fully local.

Your Choices

  • Use local mode (default) to avoid sending text to any server.
  • Enable/disable remote mode at any time in extension settings.
  • Remove the API token from settings at any time.
  • Uninstall the extension to remove stored settings from your browser.
